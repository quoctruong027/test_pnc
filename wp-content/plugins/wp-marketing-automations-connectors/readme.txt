= 1.2.0 (2020-10-09) =

* Compatible with WordPress 5.5 & 5.5.1
* Compatible with WooCommerce 4.5 & 4.6
* Added: Google Spreadsheet integration added.
* Fixed: Webhook received event has some issues, fixed.
* Fixed: Mautic token expired issue detected at some instances, fixed.
* Fixed: Mailchimp stores fetching calls has some issues, fixed.
* Fixed: Drip add or remove tags has a minor glitch, resolved.
* Fixed: Automatically syncing the connected auto-responders lists, tags etc info has a minor glitch, PHP notice was appearing, fixed now.
