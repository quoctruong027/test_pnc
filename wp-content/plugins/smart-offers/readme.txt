== Description ==

Smart Offers lets you earn more by creating a powerful sales funnel of upsells, downsells and backend offers. Show special offers during checkout, my account or on any pages.

== Installation ==

1. Ensure you have latest version of [WooCommerce](https://wordpress.org/plugins/woocommerce/) plugin installed
2. Unzip and upload the folder 'smart-offers' to your `/wp-content/plugins/` directory
3. Activate 'Smart Offers' through the 'Plugins' menu in WordPress
