== Installation ==

>[**HOW TO GUIDE**](http://www.exportfeed.com/tos) |||
>[** ALL FAQS **](http://www.exportfeed.com/faq/)

<P>
A few quick steps<P>
1) Upload to your WordPress website, and activate 'List WooCommerce Product Feed in Amazon Marketplace'.<BR>
2) Select ‘ExportFeed Amazon Market' from admin menu, Click 'Create Feed'<BR>
4) Once it refreshes, Perform attribute mapping if required<BR>
5) Select your WooCommerce Product category (or all categories) from the dropdown on the right.<BR>
6) Select the corresponding category in the destination start typing to matching Categories (i.e. auto, skin care,
baseball)<BR>
7) (Optionally) Name your feed.<BR>

<P>Click --Get Feed--

<P>
This will now redirect to the feed URL you just created - ready for submission!
