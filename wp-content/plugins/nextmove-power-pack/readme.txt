NextMove Power Pack extends NextMove capabilities more. Allow Thank You receipt tracking URL to embed on WooCommerce transactional emails. Order Timeline new component for more engagement. Coupon component display on WooCommerce order's transactional emails.

== Change log ==

= 1.2.0 =
* Added: Allow hiding HTML, Simple text, video like components based on the order status.
* Fixed: PHP files inclusion sometimes caused PHP errors on some servers only, fixed now via changing inclusion syntax.
* Fixed: Wrong coupon code in the email when payment made via PayPal, resolved now.


= 1.1.1 =
* Added: Admin notice when store owner enabled to embed Coupon/Order receipt button in the email but forgot to select the customer emails.
* Improved: License updater class improved.


= 1.1.0 =
* Added: Allow sending of coupon in email when coupon is locked. So locked coupon heading, description, button CTA etc.
* Added: The customer emails options added under both Coupon and Order receipt URL sections.
* Improved: PowerPack settings UI improved, displayed preview side by side.
